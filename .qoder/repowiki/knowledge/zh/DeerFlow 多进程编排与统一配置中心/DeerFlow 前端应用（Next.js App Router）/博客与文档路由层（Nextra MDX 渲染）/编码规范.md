- 所有页面统一通过 getI18n() 获取 locale，再经 getPreferredBlogLang 映射到博客语言代码，避免直接硬编码语言值。
- MDX 内容一律经由 useMDXComponents().wrapper 包裹，并透传 toc/metadata/sourceCode，禁止直接渲染原始 MDX 组件。
- 服务端页面以 async function Page(props) 形式接收 await props.params 与 await props.searchParams，并在找不到资源时调用 notFound() 返回 404。
- 布局文件通过 nextra-theme-docs 的 Layout 注入统一的 Header、Footer、pageMap 与 docsRepositoryBase，保持导航与仓库链接一致。
