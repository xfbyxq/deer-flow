本目录以「按功能域划分」的方式组织工作区（Workspace）页面所需的全部 React 组件：
- 布局外壳由 `workspace_shell_and_primitives` 提供（`workspace-container.tsx`、`workspace-sidebar.tsx`、`workspace-header.tsx`、`input-box.tsx` 等），作为所有子模块的宿主。
- 核心对话流由 `chats` 管理线程状态与 URL 路由，`messages` 负责消息渲染，二者通过共享的上下文/事件在右侧面板中协作。
- 辅助面板 `sidecar`、`artifacts`、`changes`、`channels`、`citations` 均以可折叠侧边栏或弹窗形式挂载到同一 Workspace 容器上，彼此之间不直接耦合，而是通过父级容器或全局状态协调可见性。
- `agents`、`settings` 作为独立入口页，同样被 `workspace-shell` 包裹渲染。
- 各子模块内部遵循统一的导出约定（`index.ts` 聚合 re-export），便于上层按需引入；跨模块共享的原语（如 `tooltip.tsx`、`copy-button.tsx`、`github-icon.tsx`、`gateway-offline-banner.tsx`）放在本层根目录供所有子模块复用。
