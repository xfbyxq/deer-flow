本模块是纯展示型 React 客户端组件集合，按职责拆分为三个文件：
- `artifact-link.tsx`：作为 Markdown 链接解析器，识别 `citation:<id>` 形式的子节点并委托给 `CitationLink`，其余普通 URL 走原生 `<a>` 渲染；通过正则 `/^citation:(.+)$/` 与 `/^https?:\/\//` 区分内部/外部链接。
- `citation-link.tsx`：基于 shadcn/ui 的 `HoverCard` + `Badge` 实现带域名的内联引用徽章，悬停弹出卡片显示完整 URL 与“Visit source”入口，使用 `new URL(url).hostname` 提取域名。
- `citation-sources-panel.tsx`：以 `<details>/<summary>` 折叠面板列出全部 `CitationSource[]`，每项含序号、标题、域名、引用计数与外链图标；内置 `CitationSourceCopyButton` 调用 `@/core/citations/sources.formatCitationMarkdownReference` 生成标准引用文本并通过 `@/core/clipboard.writeTextToClipboard` 写入剪贴板，配合 `sonner.toast` 反馈成功/失败状态。
依赖方向单向向外：仅消费 `@/core/citations/sources`、`@/core/clipboard`、`@/core/i18n/hooks` 与 UI 基础库，不暴露任何业务逻辑。
