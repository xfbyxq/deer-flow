- 所有敏感配置（`jwt_secret`、token 过期天数）统一通过 `app.gateway.auth.config.get_auth_config()` 获取，禁止硬编码。
- 对外错误采用自定义异常层次：`OIDCError` 基类派生出 `OIDCProviderError` / `OIDCValidationError` / `OIDCUserInfoMismatch`，消息直接可安全返回给 API 消费者。
- 对外部不可信字符串比较一律使用 `secrets.compare_digest` 做常量时间比较，防止时序侧信道攻击。
- 对外部网络请求统一用 try/except 捕获 `httpx.HTTPStatusError` 与 `httpx.RequestError`，并包装成领域异常抛出，不在调用层处理底层 IO 细节。
- 对外暴露的数据载体使用 frozen dataclass（`OIDCMetadata`、`OIDCIdentity`）或 Pydantic BaseModel（`TokenPayload`、`OIDCStatePayload`），保证不可变性与字段校验。
