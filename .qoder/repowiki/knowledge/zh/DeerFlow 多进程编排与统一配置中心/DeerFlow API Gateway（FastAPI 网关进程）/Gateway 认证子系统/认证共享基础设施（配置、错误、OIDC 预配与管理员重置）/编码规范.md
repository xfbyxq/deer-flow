- 配置项统一使用 Pydantic `BaseModel` + `Field(...)` 描述，并通过全局单例 + 懒加载函数（如 `get_auth_config`）对外暴露。
- 枚举类继承 `StrEnum`（`AuthErrorCode`、`TokenError`），配合 Pydantic `BaseModel` 作为结构化响应体，禁止裸字符串错误码。
- 敏感文件（`.jwt_secret`、`admin_initial_credentials.txt`）一律通过 `os.open(O_WRONLY|O_CREAT|O_TRUNC, 0o600)` 原子创建，避免先写后 chmod 的竞态窗口。
- 跨模块类型引用（如 `User`）采用运行时 `from ... import` 放在函数体内或文件末尾，以规避循环导入。
- 异步 CLI 入口使用 `asyncio.run(_run(...))` 包装协程主逻辑，并在 `finally` 中显式关闭数据库引擎。
