- 对外暴露的决策函数统一返回 (bool, reason) 元组，reason 是短标签用于日志（如 "action=opened"、"mention required for @xxx"）。
- 对磁盘 I/O 或网络 I/O 的慢路径一律通过 asyncio.to_thread 包装同步函数，保证 webhook 路由不被阻塞超过 GitHub 10s 超时。
- 进程内缓存采用「签名 + 双检锁」模式：先无锁读命中则直接返回，未命中再获取 per-key lock 后二次检查，避免并发重复计算。
- 配置读取优先读环境变量（如 GITHUB_APP_PRIVATE_KEY），其次回退到文件路径（GITHUB_APP_PRIVATE_KEY_PATH），以便热更新密钥无需重启进程。
- 模块级副作用注册通过顶层调用 register_policy() 完成，并在 __init__.py 中显式 import 该模块，使测试直接构造 ChannelManager 也能继承注册。
