- 每个路由模块独立创建 `router = APIRouter(prefix="/api", tags=[...])`，并以 `@router.get/post/put/delete` 装饰器声明端点，集中管理前缀与标签。
- 对外请求体与响应体一律使用 Pydantic `BaseModel` + `Field(..., description=...)` 描述，作为 OpenAPI schema 的唯一来源。
- 写操作的阻塞文件系统 IO（目录创建、yaml/json 写入、rmtree）通过 `asyncio.to_thread` 包装，避免阻塞 FastAPI 事件循环。
- 异常处理采用“捕获特定异常→raise HTTPException(具体状态码)”，最后用通用 `except Exception` 记录日志并返回 500，HTTPException 透传不被二次包装。
- 管理员或敏感写操作通过 `await require_admin_user(request, detail=...)` 在函数入口进行权限检查，失败即返回 403。
- 对 Skill/Agent 名称做规范化（去除换行、大小写归一、正则白名单）后再落盘，保证存储键稳定且可跨平台。
